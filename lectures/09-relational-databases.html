Exercises

  1. In your local clone of the course repository, issue this command to retrieve updates: 
    git pull origin main
  2. Within the course repository, locate the 11-datasci-sql folder and the tracking.sql SQL script.
  3. Using the RDS instance you created last week, load the SQL file into your database. You should now have a new table named "tracking".
  4. Visit Mockaroo.com and create a new table of mock data. This table should consist of 1000 records with at least 6 columns of data.
  5. Create a new table in your RDS instance matching the columns and data types of your mock table data.
  6. Your export from Mockaroo should be SQL format, but do not check the "include CREATE TABLE" option.
  7. Load your data into the database. Debug and correct errors.
  8. Should you want a greater challenge, export the Mockaroo data into CSV or JSON format and import.
