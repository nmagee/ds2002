Exercise

Using the MongoDB Atlas database cluster you have already created, complete the following steps in Python. If possible, this process should require no human intervention.

  1. Use the requests library to fetch one of the datasets below (pick one or more).
  2. Ingest and transform the data - select some of the columns to retain, and transform into a format compatible with insertion into MongoDB.
  3. Programmatically insert all transformed records from the remote dataset into your MongoDB Atlas database.

Please note that if you have loaded the sample databases into your Atlas cluster, you may want to delete them to free up storage capacity

Datasets to choose from:

  • Occupancy Detection Data - Over 20k rows, 17 columns, of temperature, humidity, light, and CO2 detection. Parse this data, inserting at least 5 columns from 1000 records into your MongoDB collection.
  • Dow Jones Industrial Average - This dataset contains weekly data for the Dow Jones Industrial Index (750 rows, 16 colums, CSV format). Extract at least 5 columns and insert into your MongoDB collection.
  • Reddit Submissions Dataset - Select a dataset (already in JSON format), decompress it, and selectively insert some of the fields from 1000 records into a MongoDB collection.
  • The Stanford IMDB Movie Review training dataset - Insert 1000 records from this dataset, using the filename as a field value and the review text as another field value.
  • Select another dataset

You may want to refer to the insert_one or the insert_many PyMongo methods to complete this task.
