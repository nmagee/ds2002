Amazon Web Services (AWS)

AWS Educate is Amazon's global initiative to provide students comprehensive resources for building skills in the cloud. It is a no-cost curriculum providing access to content, training, pathways, AWS services, and the AWS Educate Job Board with employment opportunities.

Students will be using their own AWS account for hands-on work in this course. It is very likely that you do not yet have an AWS account, so you have two options:

Create a temporary AWS Educate Starter Account by using this link. Starter Accounts require no credit card and have $100 in promotional credits. The AWS Educate Starter account offers students no-cost access to a specified, capped amount of AWS cloud resources without requiring a credit card for payment. The account provides access to most AWS Cloud services but does not allow for post-graduation portability. View the services supported with an AWS Educate Starter account.


Create a regular AWS Account and use the Free Tier by signing up here. If you have not used AWS before, creating a new account places you in the 1-year "free tier" for many AWS services. Using an AWS account allows for the greatest amount of flexibility, access to AWS services, eligibility for AWS Free Tier in year one and a larger AWS Promotional Credit amount. Services available in the Free Tier. After you have created your own AWS account, go to AWS Educate and register as a student requesting student credits for your own standalone AWS account (instead of creating a Starter Account). Within 24 hours you will be able to retrieve a coupon code for $100 that can be credited to your account.

If you already have an AWS account, simply sign up for AWS Educate as a student and request your $100 credits. They will issue you a coupon code that you can apply to your existing account.
